Financial Valuation Models

This repository contains three-statement financial models and valuation analyses for three Indian companies across different sectors.

## Companies Covered
- **Sun Pharma** (Pharmaceuticals)
- **Trent Ltd** (Retail)
- **Asian Paints** (Consumer Goods)

## Files Included
- `Sun_Pharma_Valuation.xlsx` – DCF, WACC, ratio analysis, VaR
- `Trent_Ltd_Model.xlsx` – Financial statements and valuation
- `Asian_Paints_Model.xlsx` – Financial statements and valuation

## Key Features
- 10+ years of historical financial data
- Integrated three-statement models
- DCF valuation with company-specific WACC
- DuPont ROE decomposition
- VaR analysis (historical and Monte Carlo)

## Data Sources
- Company annual reports
- Screener.in
- Moneycontrol

## Limitations
- Forecasting is linear (for learning purposes only)
- Does not model patent expiries or specific growth drivers
